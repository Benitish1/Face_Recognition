Face Recognition Using MediaPipe, LBPH & OCR

This project implements a real-time Face Recognition system built with MediaPipe for face detection and LBPH (Local Binary Patterns Histograms) for face recognition and Optical Character Recognition (OCR) for text extraction.
It allows you to train custom faces and recognize them through a webcam feed with high accuracy and speed.
